# web_petshop_demo

Este projeto é uma versão demo de um sistema web para petshops, criado com foco em simplicidade, usabilidade e apresentação profissional.

## Demo

- Site: https://petdeluxe.netlify.app/

## Visão Geral

- Front-end estático em HTML, CSS e JavaScript
- Catálogo de produtos com filtros por tipo e animal
- Layout responsivo e fácil de navegar

## Estrutura

- `index.html`: página principal
- `assets/css/style.css`: estilos
- `assets/js/script.js`: interações básicas
- `assets/imagem/`: imagens de produtos

## Como Executar Localmente

- Clone o repositório e abra `index.html` diretamente no navegador
- Ou sirva com qualquer servidor estático (ex.: `npx serve`)

## Deploy

- Hospedado via Netlify: https://petdeluxe.netlify.app/
